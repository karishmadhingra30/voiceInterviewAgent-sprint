  * Connections to famous investors or public figures

- Only extract what is explicitly present in the profile data
- Never invent or infer facts not in the data
- If a field has nothing to report return empty list or empty string
- Never include raw profile data in your response

  BAD:  "experienced entrepreneur"
  GOOD: "Senior Policy Advisor for Tech Inclusion, Obama White House"

Respond ONLY with valid JSON. No markdown, no explanation, no code fences.

those are handled by the system):
